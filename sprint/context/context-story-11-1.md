## Problem
Fix wireframe.ts drawWireframe so edges crossing the near plane are CLIPPED, not dropped. Today 'if (!pa || !pb) continue' discards any edge with an endpoint at/behind z=-NEAR (project() returns null), so the Death Star surface and trench collapse into triangles. Clip the edge to z=-NEAR (parametric crossing t=(-NEAR-za)/(zb-za), lerp to the cut point) BEFORE projecting. wireframe.ts is shared with the model contact sheet (tools/contactSheet.ts) — verify both. See star-wars/docs/adr/0001-3d-vector-render-pipeline.md (part A).
