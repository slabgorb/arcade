## Problem
Cluster C11 (GP-003). Small, cheap, and it UNBLOCKS regression-testing every other story in this epic — do it early. main.ts:467 calls createRng((Date.now() + kills)) to spawn every wave, INSIDE the sim-step path, and main.ts:349 seeds the blimp RNG from the wall clock too. The ROM's RANDOM (RBARON.MAC:6193) is a pure software LFSR with no clock input. Two consequences: the sim is non-deterministic (same-seed replay and same-seed regression tests are impossible — which is exactly what the rest of this epic needs), and it violates the core/shell boundary, which forbids the deterministic sim from reading wall-clock time.
