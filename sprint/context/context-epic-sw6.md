# Epic sw6 Context

## Title
Star Wars — ROM audio fidelity (the music that has never played)

## Overview
Star Wars ships a complete, correct music path that has never made a sound: sw3-5 wired the core phase-edge MusicEvent -> main.ts startLoop -> @arcade/shared/audio music channel, but the four .wav assets it points at were never produced and 404 on R2 (the sfx/ prefix, by contrast, is populated). The shared engine degrades silently at every failure path, so a missing asset is indistinguishable from working code — the game has simply been quiet since the feature shipped. Asset production was explicitly scoped out of sw3-5 and flagged by TEA, Dev and the Reviewer as a non-blocking Delivery Finding; no follow-up story was ever filed. This epic is that follow-up.
