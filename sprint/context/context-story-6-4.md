## Problem
Adopt the 'Vector Battle' arcade vector font (VectorBattle-e9XO.ttf — ck!/freakyfonts, 1999; Freeware Non-Commercial) for HUD and framing-screen text, replacing the current default-font rendering, to match the glowing-vector aesthetic. SHELL/RENDER-ONLY — no src/core/ impact. Load via the FontFace API / @font-face and ship the 29KB TTF as a static asset honoring Vite base:'/tempest/'. The face is CAPS-ONLY, so render text uppercase. Thin strokes get fragile at small sizes — render HUD at >=18-20px and/or reduce glow blur. License cleared for this non-commercial project; a commercial license must be purchased from the designer if the project ever becomes commercial. Full UX spec: docs/ux/2026-06-27-tempest-arcade-feel-reference.md (Typography section).
