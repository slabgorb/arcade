## Problem
Carryforward from rb2-1 (flight model). Enemy render inherits the flight-model → camera scale factors in red-baron/src/core/flight.ts: toAttitude/toEye map ROM units → radians/world via ROLL_SCALE=π/1024 (0x200 quadrant → 0x100=45° max bank), PITCH_SCALE=π/512, YAW_SCALE=π/1024, ALT_TO_Y=1/4 (I4YPOS x4 → world eye Y) — draw enemies in the SAME world frame these produce. Enemy bank proportional to turn-rate (this title) should reuse the PFROTN=PLDELX×8 coupling from flight.ts. OPEN (TEA/Reviewer finding, rb2-1): the PFROTN bit-format angle (XXX XQQA AAAA AA.FF, findings §2) is only a scalar in flight.ts; if exact projection is needed, add an explicit ROM-angle→radian converter rather than the scalar.

## Technical Approach

**Render Pipeline:**
1. Define an Enemy interface mirroring FlightState (turnRate, pitchRate, altitude, heading) — enemies are aircraft with identical dynamics.
2. Transform enemy position from world space to camera space using the shared Math Box (`@arcade/shared/math3d`), reusing the scale factors from flight.ts:
   - `ROLL_SCALE = π/1024` (map ROM angle → radians)
   - `PITCH_SCALE = π/512`
   - `YAW_SCALE = π/1024`
   - `ALT_TO_Y = 1/4` (I4YPOS units → world eye height)
3. Compose enemy Attitude using the same `flightView(attitude, eye)` camera matrix that drives the horizon.
4. For LOD selection: compute depth (distance to camera). Use `DB.MAP + DB.MAR + DB.LNS` (42-vertex full biplane) for near distances; use `DB.MAR` only (29-vertex drone) for far.
5. Decode the connect-lists via `decodeOp()` from topology.ts, project each edge through the scene substrate (`sceneProjection()` + `projectSegment()`), and stroke visible segments (pen-down ops).

**Bank Coupling:**
- Enemy bank should follow `pfrotn(enemy.turnRate)` — the same `PLDELX × 8` coupling as the player. This derives directly from the roll component of the Attitude.
- The ROM PFROTN bit format (XXX XQQA AAAA AA.FF) is only a ROM internal; flight.ts already abstracts it as a scalar, so the bank angle is ready to pass to the attitude.
