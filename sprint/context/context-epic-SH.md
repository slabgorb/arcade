# Epic SH Context

## Title
Shared arcade library — extract cross-repo duplication into @arcade/shared

## Overview
Extraction epic arising from the 2026-07-04 cross-repo survey and ratified by docs/adr/0001-shared-code-strategy.md (Accepted). Shared code lives in a new dedicated repo `arcade-shared`, published as scoped package `@arcade/shared` with subpath exports (/math3d, /rng, /highscore, /loop), consumed by each game + the lobby as a version-pinned git-URL npm dependency (no registry). Eligibility bar: only code that is byte/algorithm-identical across >=2 games is extracted — game-specific render pipelines, sim bodies, and input maps stay put. Sequence proves the pipe cheaply first (math3d + rng are byte-identical and comment-flagged 'port, don't share') before harder migrations. NOTE: the vector-font strategy is a SEPARATE decision (future ADR-0002) — two incompatible approaches (Vector Battle TTF vs tempest's ROM stroke font) — and is NOT in this epic; it gates bz2-2/A2-2 independently.
